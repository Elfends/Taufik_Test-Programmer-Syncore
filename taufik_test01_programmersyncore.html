1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ? Diperuntukan untuk menjadi (cetakan)struktur dasar pengembangan, sehingga dapat mempermudah pendokumentasian dan pemahaman struktur. Meningkatkan rate scalability, continuity pengembangan, dan mengurangi unconsistency dalam coding. 2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach? if else : menentukan tindakan yang harus dilakukan bila terjadi kondisi [if] dan tindakan bila kondisi [if] tidak terjadi. while : mengulang proses yang sudah ditentukan selama kondisi [while] masih terpenuhi. while do : melakukan proses yang didefinisikan selama kondisi [while] terpenuhi. swicth case : menentukan tindakan yang dilakukan menurut kesamaan nilai [switch] dan [case] yang terjadi. for : proses pengulangan dilakukan sebanyak iterasi yang didefinisikan. for each : pengulangan dilakukan selama anggota dari list masih ada. 3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ? constructor : menginisiasi dan mendefinisikan [object] dari [class] method : adalah suatu proses yang dilakukan pada data dengan struktur tertentu. class : adalah pengorganisasian data dan perilaku(method) menjadi satu entitas. object : merupakan list(array) dengan data majemuk(lebih dari 1 type data) menggunakan [key] sebagai indexing, dan dapat didefinisikan oleh [class]. variable : penamaan alokasi data untuk menyimpan nilai yang dapat dipanggil, digunakan dan diubah. parameter : [variable] yang digunakan untuk [passing] nilai kedalam sebuah [function] atau [procedure]. function : serangkaian tindakan yang dipanggil dan memberikan nilai balikan (return value). procedure : serangkaian tindakan terhadap data (yang tidak memberikan balikan) yang dapat menentukan proses berikutnya. 4. Perbedaan method=”Post” dan “Get” ? post : -mengirim data untuk diproses yang biasanya akan disimpan ke database. -tidak muncul pada bilah url. -lebih secretive. get : -mengirim parameter untuk mendapatkan data dari database. -muncul pada bilah url dengan tanda [?] -dapat melihat parameter yang digunakan untuk function pada url yang dituju 5.
